# Random Password Generator

## Objective
This project generates strong random passwords based on user preferences.

## Tools Used
Python

## Steps Performed
1. Asked the user for password length.
2. Asked whether to include uppercase, lowercase, numbers, and symbols.
3. Used Python libraries like random and string.
4. Generated a secure random password.

## Outcome
The program creates strong and customizable passwords for better security.
